RED HOOD

Developed a fully functional 2D side-scroller game entirely using Blueprints in Unreal Engine 5. The main character is built using Sprites and Flipbooks, with PaperZD managing the animation logic in a clean and modular way. The level design was crafted using Tilesets, giving it a rich and visually cohesive aesthetic. For enemy AI, I used a Skeletal Mesh and implemented a robust Behavior Tree system to handle core behaviors like patrolling, detecting the player, attacking, and dying. This setup allowed me to create responsive, believable enemy interactions while keeping the game lightweight and efficient. The project showcases my understanding of 2D pipeline integration, animation systems, and AI logic within Unreal Engine.

GUNCRAFT

GunCraft is a fast-paced third-person multiplayer shooter built in Unreal Engine with full Blueprint implementation. The game supports online multiplayer through Steam integration, enabling global matchmaking and smooth player connections. Players can choose from a diverse arsenal including Pistols, SMGs, Rifles, Shotguns, Snipers, Rocket Launchers, and Grenades. Each weapon has unique attributes like fire rate, damage, and spread, offering distinct combat strategies. Core mechanics are powered by the Enhanced Input System, ensuring responsive and fluid controls. Buffs and pickups like Speed Boosts, Jump Buffs, Health, Shields, and Ammo create dynamic gameplay moments. Three competitive modes are available: FreeForAll, Team Deathmatch, and CaptureTheFlag, each demanding different tactics. Lag Compensation via Server-Side Rewind ensures fair hit registration for players with ping below 120ms, improving online play balance.
